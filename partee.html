The Process

This game is built using the Angular Typescript Framework and all illustrations were hand drawn by myself! This game is still a work in progress as I work to add in the "Par-Tee Time" functions that can be enabled on the leader create screen. The goals of the "Par-Tee Time" function would be to punish the players who click bingo tiles at random by wiping their board when a non-matching tile is clicked, offer an opportunity to gather more free spaces via an embedded challenge, and increase competition by allowing players to punish one another if the embedded free space challenge is completed successfully! "Par-Tee Time" would elevate the game beyond just a simple round of Bingo and make the game more challenging for individual players!
